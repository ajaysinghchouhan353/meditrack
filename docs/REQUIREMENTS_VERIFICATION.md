# Requirements Verification - MediTrack

## Verified Coverage

- **OOP**: Encapsulation, inheritance, polymorphism, and abstraction are present.
- **Design Patterns**: Singleton, Factory, Strategy, Observer, Repository, Service Layer, and Template Method are implemented.
- **Collections and Streams**: The code uses generics, streams, and thread-safe collections.
- **Persistence**: Data can be exported and restored through CSV and serialization support.
- **Validation**: Input validation and custom exceptions are in place.
- **Testing**: The project includes a manual test runner and compiles cleanly with Maven.

## Notes

- The implementation is intentionally console-based.
- Advanced UI and database integrations are out of scope for this project.
